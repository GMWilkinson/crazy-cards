# Crazy Cards

#### Technical test.
Allows customers to enter their details and review credit cards that are applicable to their credit status.

## Installation

### `yarn`

```zsh
yarn
```


## Run

### `Start Server`

```zsh
yarn start:server
```
### `Start Client`

```zsh
yarn start
```

### `Tests`

```zsh
yarn test
```
